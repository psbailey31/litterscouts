# Components Directory

This directory contains all React components for Litter Scouts.

## Structure

Components are organized by feature:
- `auth/` - Authentication components (SignIn, SignUp, ProtectedRoute, UserProfile)
- `layout/` - Layout components (Header, Footer, Navigation)
- `map/` - Map-related components (MapView, MapFilters)
- `report/` - Report-related components (PhotoUpload, ReportForm, ReportDetail)
- `events/` - Event management components (coming soon)
- `analytics/` - Analytics and dashboard components (coming soon)
- `common/` - Shared/reusable components (coming soon)

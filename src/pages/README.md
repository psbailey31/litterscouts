# Pages Directory

This directory contains page-level components that represent different routes in the application.

## Planned Pages

- `HomePage.tsx` - Landing page with map view
- `ReportPage.tsx` - Report submission page
- `AnalyticsPage.tsx` - Analytics dashboard
- `EventsPage.tsx` - Event listing and management
- `EventDetailPage.tsx` - Individual event details
- `ProfilePage.tsx` - User profile page
- `NotFoundPage.tsx` - 404 error page

# Services Directory

This directory contains service modules for API communication and external integrations.

## Planned Services

- `api.ts` - Base API client (already created)
- `reportService.ts` - Report-related API calls
- `eventService.ts` - Event-related API calls
- `analyticsService.ts` - Analytics API calls
- `userService.ts` - User profile API calls
- `exifService.ts` - EXIF metadata extraction
- `externalApiService.ts` - Weather, tides, water quality APIs
